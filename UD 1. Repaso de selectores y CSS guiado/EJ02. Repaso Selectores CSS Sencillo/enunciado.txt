A partir del código HTML proporcionado, añadir las reglas CSS necesarias para que la página resultante tenga el 
mismo aspecto que el de la siguiente imagen.

Aunque la propiedad que modifica el color del texto se explica detalladamente en los próximos capítulos,
en este ejercicio solamente es preciso conocer que la propiedad se llama color y que como valor se puede indicar
directamente el nombre del color.

Los nombres de los colores también están estandarizados y se corresponden con el nombre en inglés de cada color.
En este ejercicio, se deben utilizar los colores: teal, red, blue, orange, purple, olive, fuchsia y green.
